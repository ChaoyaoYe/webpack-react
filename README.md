## 环境要求
```  
nodejs: v4.4.7+
```

## 安装依赖

```    
npm install
```
```
// 淘宝源
npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install
```

## 开始
1. `npm start`
2. 用浏览器打开 http://localhost:3000

## 其他指令
```
// 更新依赖
npm run update

// 打包
npm run build 
```
